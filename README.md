Synopsis of Project: We will build an LLM "Audit Trail", addressing the Governance and Security challenges of integrating AI into company workflows. Many companies face a "black box" problem, where it is difficult to track employee AI usage to prevent trade secret leakages or ensure that biased or hallucinated advice is not passed to clients. Our project aims to build a real-time AI proxy and observability layer, which will redact PII, track version control system prompts, and create company audit logs which will serve as a dashboard to understand how AI is being used across companies.
